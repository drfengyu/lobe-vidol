# 智能体功能介绍视频制作指南

本文档说明如何用 Lobe Vidol 角色制作智能体功能介绍视频，实现角色边讲解边做「指来指去」的手势，与界面操作对应。

---

## 一、整体思路

1. **角色讲解**：用视频模式（Camera Mode）让角色通过 TTS 朗读介绍文案
2. **手势动作**：在讲解过程中播放「 pointing / 手势」类动作，与 UI 操作对应
3. **录制方式**：用 OBS 等录屏软件同时录制角色画面和界面操作

---

## 二、可用的「指/手势」类动作

### 2.1 预设动作（对话时自动触发）

情感分析会从以下预设中选择动作，适合在介绍文案中通过语气引导：

| 动作 ID | 说明 | 适合场景 |
|---------|------|----------|
| `female_greeting` | 招手打招呼 | 开场、指向观众 |
| `female_appeal` | 吸引/展示 | 强调重点、指向屏幕 |
| `number_meter_1` | 数字 1 手势（伸出一指） | **最接近「指」的动作** |
| `number_meter_2` | 数字 2 手势 | 列举第二点 |
| `number_meter_3` | 数字 3 手势 | 列举第三点 |
| `female_happy` | 开心 | 轻松讲解 |

### 2.2 动作列表中的手势（需手动点击）

在 **设置 → 动作** 中，筛选 **动作类别：日常** 可找到：

| 名称 | 性别 | 描述 |
|------|------|------|
| **Angry Point** | 男性 | Quickly Pointing Angrily Forward（快速向前指） |
| **Happy Hand Gesture** | 男性 | Happy Forward Hand Gesture（开心向前挥手势） |
| **Angry Gesture** | 男性 | Angrily Pumping Fists Forward |

> 若角色为女性，可尝试上述男性动作，部分 VRM 骨骼兼容；或使用女性动作如 `Happy`、`Excited` 配合表情。

## 三、制作流程

### 步骤 1：准备介绍文案

按功能点分段，并在需要「指」的地方加入语气词，方便情感分析选到合适动作：

```
大家好，我是朗科智语。今天为大家介绍 Lobe Vidol 的智能体功能。

【第一点】首先，点击这里可以创建新的智能体对话。
【第二点】在设置中，你可以配置不同的语言模型。
【第三点】最后，智能体支持多轮对话和记忆功能。
```

在文案中适当加入「首先」「第二」「第三」等，有助于触发 `number_meter_1/2/3` 手势。

### 步骤 2：进入视频模式并布置画面

1. 启动项目：`npm run dev`
2. 进入 **视频模式**（Camera Mode）
3. 选择要出镜的角色
4. 打开 **设置**（齿轮图标）→ **动作**，筛选 **日常**，找到 `Angry Point`、`Happy Hand Gesture` 等，先点一下确认动作效果

### 步骤 3：录制方案

**方案 A：纯 TTS + 情感分析（推荐）**

- 在底部输入框输入介绍文案（可分段发送）
- 系统会根据情感分析自动选择表情和动作
- 若希望更多「指」的动作，可在文案中多用「第一」「第二」「这里」「那里」等词

**方案 B：TTS + 手动切换动作**

- 发送文案让角色开始讲解
- 在讲解过程中，打开 **设置 → 动作**，在合适时间点点击 `Angry Point` 或 `Happy Hand Gesture` 等
- 需要提前熟悉每个动作的时长，以便卡点

**方案 C：分屏录制**

- 左侧：角色讲解（视频模式全屏或单独窗口）
- 右侧：智能体界面操作演示
- 用 OBS 等软件同时录制两个画面，后期合成

### 步骤 4：使用 OBS 录制

1. 安装 [OBS Studio](https://obsproject.com/)
2. 添加 **窗口捕获** 或 **显示器捕获** 源
3. 调整布局：角色在左/右，界面在另一侧
4. 点击「开始录制」，按脚本操作并讲解
5. 录制结束后在 OBS 中停止并保存

## 四、增强「指」动作的两种方式

### 方式 1：扩展情感分析提示词（推荐）

在 `src/chains/emotionAnalysis.tsx` 中，可在 system prompt 里加入说明，让模型在「介绍、讲解、指出」时优先选择 `number_meter_1`、`female_appeal` 等：

```ts
// 在 chainEmotionAnalysis 的 system content 中补充：
当内容涉及「介绍」「指出」「这里」「那里」「第一点」「第二点」时，
优先使用 number_meter_1、number_meter_2、number_meter_3 或 female_appeal。
```

### 方式 2：将「指」动作加入预设映射

若希望情感分析能直接使用 `Angry Point` 等动作，需要：

1. 在 `motionPresetMap.ts` 中新增预设（如 `female_point`）
2. 填入对应动作的 URL（从 `Motion/index.json` 中获取）
3. 在 `emotionAnalysis` 的 prompt 中把新预设加入可选动作列表

## 五、注意事项

1. **性别匹配**：女性角色优先用 `female_*`，男性用 `male_*`，否则可能出现骨骼错位
2. **网络**：动作资源来自 r2.vidol.chat，首次播放需下载，建议提前在动作列表中点一遍预加载
3. **口型同步**：TTS 会自动驱动唇形，无需额外配置
4. **背景**：可在 **设置 → 背景** 中选择合适的虚拟背景，使画面更专业

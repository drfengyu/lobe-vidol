# Lobe Vidol 角色动作与表情参考

本文档整理了 Lobe Vidol 中角色可用的**动作（Motion）**和**表情（Expression）**，供配置触摸交互、情感分析或 Dify 智能体时参考。

---

## 一、动作（Motion）

动作用于控制角色的身体动画，在 `motionPresetMap` 中定义。每个动作有唯一 ID，配置时使用 ID 作为 `motion` 的值。

### 1.1 女性动作

| ID | 英文名 | 中文说明 | 格式 |
|----|--------|----------|------|
| `female_happy` | Female/Happy | 开心 | FBX |
| `female_angry` | Female/Angry | 生气 | FBX |
| `female_greeting` | Female/Greeting | 招手打招呼 | FBX |
| `female_appeal` | Female/Appeal | 吸引/展示 | VMD |
| `female_cover_undies` | Female/Cover Undies | 捂裙子（害羞） | VMD |
| `female_cover_chest` | Female/Cover Chest | 捂胸 | VMD |
| `female_kiss` | Female/Kiss | 亲吻 | VMD |
| `female_stand` | Female/Stand | 站立 | VMD |
| `female_stand_mix` | Female/Stand Mix | 站立混合 | VMD |

### 1.2 男性动作

| ID | 英文名 | 中文说明 | 格式 |
|----|--------|----------|------|
| `male_happy` | Male/Happy | 开心 | FBX |
| `male_angry` | Male/Angry | 生气 | FBX |

### 1.3 通用动作

| ID | 英文名 | 中文说明 | 格式 |
|----|--------|----------|------|
| `idle` | Female/Stand | 待机/空闲 | VMD |

### 1.4 数字手势（Number Meter）

| ID | 英文名 | 说明 |
|----|--------|------|
| `number_meter_1` | Number Meter 1 | 数字 1 手势 |
| `number_meter_2` | Number Meter 2 | 数字 2 手势 |
| `number_meter_3` | Number Meter 3 | 数字 3 手势 |
| `number_meter_4` | Number Meter 4 | 数字 4 手势 |
| `number_meter_5` | Number Meter 5 | 数字 5 手势 |

## 二、表情（Expression）

表情基于 VRM 1.0 标准 BlendShape，用于控制角色的面部表情。配置时使用 ID 作为 `expression` 的值。

### 2.1 口型（Mouth）

用于嘴唇形状，通常由 Lip Sync 自动控制，也可手动指定：

| ID | 说明 |
|----|------|
| `aa` | 嘴型「啊」 |
| `ih` | 嘴型「一」 |
| `ou` | 嘴型「呜」 |
| `ee` | 嘴型「诶」 |
| `oh` | 嘴型「哦」 |

### 2.2 情绪表情

| ID | 说明 |
|----|------|
| `neutral` | 中性/自然 |
| `happy` | 开心 |
| `angry` | 生气 |
| `sad` | 难过 |
| `relaxed` | 放松 |
| `surprised` | 惊讶 |

### 2.3 眨眼

| ID | 说明 |
|----|------|
| `blink` | 双眼眨眼 |
| `blinkLeft` | 左眼眨眼 |
| `blinkRight` | 右眼眨眼 |

### 2.4 视线（Look At）

| ID | 说明 |
|----|------|
| `lookUp` | 向上看 |
| `lookDown` | 向下看 |
| `lookLeft` | 向左看 |
| `lookRight` | 向右看 |

### 2.6 触摸交互中可配置的表情

在角色编辑 → 外壳/触摸 中，每个触摸动作可配置的表情为：

- `neutral` - 自然
- `happy` - 开心
- `angry` - 生气
- `sad` - 难过
- `relaxed` - 放松
- `surprised` - 惊讶
- `blink` - 眨眼
- `blinkLeft` - 左眼眨眼
- `blinkRight` - 右眼眨眼

## 三、使用示例

### 3.1 触摸动作配置（TouchAction）

```json
{
  "text": "主人，不要摸这里啦~",
  "expression": "happy",
  "motion": "female_cover_chest"
}
```

### 3.2 情感分析输出（JSON）

```json
{
  "expression": "happy",
  "motion": "female_greeting",
  "reason": "用户表达了问候，适合开心表情和招手动作"
}
```

### 3.3 Dify 智能体返回格式建议

```json
{
  "expression": "happy",
  "motion": "female_happy"
}
```

## 四、注意事项

1. **性别匹配**：女性角色使用 `female_*` 动作，男性角色使用 `male_*` 动作，否则可能出现骨骼错位。
2. **动作格式**：FBX 为完整骨骼动画，VMD 为 MikuMikuDance 格式，通常兼容 VRM。
3. **口型**：`aa` 等口型在说话时由 Lip Sync 自动驱动，无需在情感分析中指定。
4. **表情与动作组合**：可自由组合任意表情 + 任意动作，如 `happy` + `female_greeting` 表示开心地招手。
